# Proyecto-de-grado
# Simulador de Escenarios para Contact Center

## 🚀 Instrucciones de Instalación y Despliegue Local

Sigue estos pasos para levantar el entorno de desarrollo en tu máquina local:

### 1. Clonar el repositorio y abrir el proyecto
Abre tu editor de código (preferiblemente **Visual Studio Code**) e ingresa a la carpeta raíz del proyecto.

### 2. Instalar dependencias
Abre la terminal integrada (`Ctrl + Ñ`) y ejecuta el gestor de paquetes para instalar los módulos necesarios:
```bash
npm install
```
### 3. Arrancar el Servidor Local con Vite
Para compilar los archivos de TypeScript/JavaScript y levantar la interfaz de desarrollo, ejecuta en la terminal:
```bash
npm run dev
```
